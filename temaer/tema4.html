LÆRING

I dette tema lærte vi at udvikle brugergrænseflader med fokus på både design og funktionalitet. Herunder, darkmode, user input mm. Gennem projektet arbejdede vi med HTML, CSS og JavaScript for at skabe en interaktiv hjemmeside. Vi lærte også at anvende Adobe Illustrator til at udvikle grafiske elementer og animationer, som bidrog til en mere engagerende brugeroplevelse.
